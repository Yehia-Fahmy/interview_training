# Case Studies

This directory contains real-world system design case studies that demonstrate how large-scale systems are architected in practice.

## Case Studies

1. **Case Study 01: ChatGPT** - How OpenAI serves ChatGPT at scale
2. **Case Study 02: Netflix Recommendations** - Netflix's recommendation system architecture
3. **Case Study 03: Uber Real-time** - Uber's real-time matching and pricing system

## How to Use

1. Read each case study to understand real-world architectures
2. Analyze the design decisions and trade-offs
3. Compare with your own designs
4. Learn from production patterns and best practices
5. Use as reference when designing similar systems

## Learning Objectives

- Understand how real systems handle scale
- Learn production patterns and best practices
- See trade-offs made in practice
- Get inspiration for your own designs
